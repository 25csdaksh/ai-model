# 🤖 Daksh AI Chatbot - Python Code Base

આ ડિરેક્ટરીમાં તમારું **Python AI Chatbot** સંપૂર્ણપણે તૈયાર છે!

---

## 📁 ફાઇલોની યાદી (File Structure)

1. **`chatbot.py`**  
   - ટર્મિનલ / કમાન્ડ પ્રોમ્પ્ટમાં ચાલતું સુંદર **Python CLI AI Chatbot**.
2. **`app_server.py`**  
   - Python નું કસ્ટમ **Web Server (HTTP Backend)** જે તમારા Web UI સાથે કનેક્ટ થાય છે.
3. **`index.html`**, **`style.css`**, **`app.js`**  
   - મોડર્ન અને આકર્ષક **Web Chatbot UI Interface**.

---

## 🚀 સ્ક્રિપ્ટ ચલાવવાની રીત (How to Run)

### 1. ટર્મિનલમાં ચલાવવા માટે (Python Terminal Chatbot)

કમાન્ડ પ્રોમ્પ્ટ / ટર્મિનલ ઓપન કરી નીચેનો કમાન્ડ રન કરો:

\`\`\`bash
python chatbot.py
\`\`\`

> 💡 **ટીપ:** તમારી પાસે Gemini API Key હોય તો તે એન્ટર કરી શકો છો, અથવા ઓપ્શનલ તરીકે ખાલી મૂકીને પણ ટેસ્ટ કરી શકો છો.

---

### 2. વેબ બ્રાઉઝરમાં ચલાવવા માટે (Python Web App Server)

નીચેનો કમાન્ડ રન કરીને Python સર્વર સ્ટાર્ટ કરો:

\`\`\`bash
python app_server.py
\`\`\`

ત્યારબાદ તમારા બ્રાઉઝરમાં **`http://localhost:8000`** ખોલો!

---

## 🔑 Gemini API Key કેવી રીતે મેળવવી?
1. [Google AI Studio](https://aistudio.google.com/app/apikey) પર જાઓ.
2. મફતમાં **Create API Key** બટન પર ક્લિક કરો.
3. કી કોપી કરીને તમારા Python Chatbot માં અથવા UI સેટિંગ્સમાં પેસ્ટ કરો.
